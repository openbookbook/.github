# bookbook
Let your book club vote for its next book using ranked choice voting!

bookbook is inspired by simple but amazing organizing tools like [when2meet](https://www.when2meet.com/). No emails, no sign ups, passwords are optional.

## technology
* backend
  - NestJS framework
  - database uses MySQL
  - hosted with AWS RDS and App Runner using a Docker image
* front-end
  - React
  - full-typed despite using javascript files because [JSDoc is TypeScript](https://culi.bearblog.dev/jsdoc-is-typescript/)
